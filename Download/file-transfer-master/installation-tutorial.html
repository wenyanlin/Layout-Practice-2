安裝教學

Step 1：下載檔案

進入家裡蹲官方網站，找到路徑「首頁 > 下載中心 > 傳檔大師」，點選下載按鈕。

Step 2：安裝檔案

開啟下載好的「OrderManager_Setup-4.2.24.0.msi」檔案開始進行安裝。

Step 3：檢查必要的安裝

安裝傳檔大師時將會自動檢查您的電腦是否有安裝 Microsoft Visual VC++ 2015 以上的版本。

  • 若尚未安裝 Microsoft Visual VC++：跳到Step3-1
  • 若已安裝 Microsoft Visual VC++：跳到Step3-2
Step 3-1：下載 Microsoft Visual VC++ 2015

安裝傳檔大師時將會出現下圖，請點擊[是(Y)]，接著點擊[執行(R)]，將會進行下載程序。

Step 3-1-2：安裝 Microsoft Visual VC++ 2015

下載後執行「vc_redist.x86.exe」，
將會出現右圖，請點擊[執行(R)]進行安裝。

Step 3-1-3：接受授權條款

完成上述步驟後將出現右圖，
請勾選「我同意授權條款及條件(A)」，
再點擊[安裝(I)]以進行安裝。

Step 3-1-4：允許程式進行安裝

完成上述步驟後將出現右圖，
請點擊[是(Y)]，允許程式進行安裝。

Step 3-1-5：等待程式安裝完成

完成上述步驟後將出現右圖，請耐心等待直到安裝完成。

Step 3-1-6：關閉安裝完成的畫面

完成上述步驟後請點擊[關閉]。

Step 3-2：安裝 Microsoft .NET Framework 4.5.2 以上的版本

再次執行「OrderManager_Setup-4.2.24.0.msi」，安裝時將會自動檢查您的電腦是否有安裝 Microsoft .NET Framework 4.5.2 以上的版本。

  • 若尚未安裝 Microsoft .NET Framework 4.5.2 以上的版本：跳到Step 4
  • 若已安裝 Microsoft .NET Framework 4.5.2 以上的版本：跳到Step 3-2-2
Step 3-2-2：下載 Microsoft .NET Framework 4.5.2 以上的版本

安裝傳檔大師時將會出現下圖，請點擊[是(Y)]，接著點擊[執行(R)]，將會進行下載程序。

Step 3-2-3：允許程式進行安裝

完成上述步驟後將出現右圖，
請點擊[是(Y)]，允許程式進行安裝。

Step 3-2-4：接受授權條款

完成上述步驟後將出現右圖，
請勾選「我已閱讀並接受授權條款(A)」，
再點擊[安裝(I)]以進行安裝。

Step 3-2-5：等待程式安裝完成

完成上述步驟後將出現右圖，請耐心等待到安裝完成。

Step 3-2-6：關閉安裝完成的畫面

完成上述步驟後請點擊[完成]。

Step 4：安裝傳檔大師

再次執行「OrderManager_Setup-4.2.24.0.msi」。

Step 4-1：允許程式進行安裝

執行「OrderManager_Setup-4.2.24.0.msi」後將出現左側畫面，請耐心等待。
執行完左側畫面後將出現右側畫面，請點擊[是(Y)]按鈕，允許進行安裝。

Step 4-1-2：等待程式安裝完成

允許進行安裝後，請耐心等到出現左側畫面，並點擊[關閉(C)]以結束安裝作業。
與此同時系統會自動啟動傳檔大師，將出現右側畫面，代表傳檔大師已安裝完成。
